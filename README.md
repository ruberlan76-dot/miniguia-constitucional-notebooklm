# miniguia-constitucional-notebooklm
Este projeto tem como objetivo explorar o uso da Inteligência Artificial, por meio do NotebookLM, como ferramenta de aprendizagem ativa aplicada à preparação para concursos públicos.

O foco está na construção de um método de estudo eficiente, reutilizável para diferentes disciplinas e editais.

Objetivos:
- Desenvolver autonomia no uso da IA;
- Melhorar a forma de estudar;
- Criar resumos estratégicos;
- Aprender a fazer bons prompts;
- Montar material de revisão.

---

## 📚 Curadoria de Fontes

Fontes utilizadas:

- Constituição Federal de 1988
- Apostilas de concursos públicos
- PDFs de disciplinas básicas
- Sites educacionais
- Questões comentadas de provas anteriores

## 📘 Glossário

- Lei seca: texto literal da lei
- Pegadinha: alternativa que confunde
- Interpretação: entender o texto
- Lógica: raciocínio estruturado
